# VintageWisdom Docs

本目录按“上线交付”视角整理，建议阅读顺序如下：

1. `deployment/QUICK_DEPLOY.md`
2. `deployment/DEPLOYMENT_CHECKLIST.md`
3. `api/API_SPECIFICATION.md`
4. `development/QUICK_START_GUIDE.md`

文档分区：

- `architecture/`：系统结构与架构说明
- `design/`：界面与交互设计稿
- `deployment/`：部署、Nginx、上线检查清单
- `development/`：开发与优化指南
- `roadmap/`：后续规划与版本路线
- `api/`：接口说明
